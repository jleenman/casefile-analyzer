# Prompt History

## Beleid

De map `prompts/` wordt volledig door Codex beheerd. Codex mag prompts schrijven, aanpassen, verwijderen, opsplitsen, samenvoegen en vervangen wanneer dit de ontwikkelpipeline verbetert.

Iedere wijziging wordt hier geregistreerd met:

- datum;
- prompt-id;
- wijziging;
- reden;
- opvolging.

## Archiveringsregel
