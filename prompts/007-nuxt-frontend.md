# Prompt 007 - Nuxt Frontend

## Doel

Maak pas in deze stap een minimale Nuxt-demo voor upload, status en rapportweergave.

## Context

Tot deze prompt mag er geen Nuxt-installatie of applicatiecode bestaan.

## Inputbestanden

- `docs/mvp-scope.md`
- `docs/architecture.md`
- `docs/privacy-security.md`
- `examples/synthetic-report.md`

## Outputbestanden

- Nuxt frontendbestanden.
- Documentatie-update over lokale start en demo-flow.

## Acceptatiecriteria

- Geen gebruikersaccounts.
- Demo kan synthetische data tonen.
- UI maakt duidelijk dat dit geen juridisch advies is.
- Privacy, bronverwijzingen en controleerbaarheid zijn leidend.
- Voeg tests of validatiecriteria toe waar code wordt toegevoegd.
- Eindig met een kleine commit of duidelijke wijzigingssamenvatting.
